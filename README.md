# gnn-hydrogel-structure
Designed and trained a graph neural network (GNN) to predict network properties from molecular graph data

# Problem
Full 3D molecular simulations of hydrogel networks are slow, taking approximately 5 minutes per structure. This limits how many molecules can be modeled.

# Method
Built a data pipeline to clean and structure output from hundreds of 3D molecular simulations. Represented molecular structure as graph data. Trained a graph neural network to predict structural properties directly from molecular geometry.
